Just Wait test tool

What does it do?

Executes a function after a specified amount of time (timeout from now) from the beginning of an Ajax request.
If the AJAX requests ends before the specified amount of time, the function will never be executed.

The tool below allows you to test that functionality (see jqXHR -> wait event in the status section).

See the documentation page for more information.
